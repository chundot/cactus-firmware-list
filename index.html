Cactus Unofficial Website

Unofficial support for cactus firmware update.

Firmwares and softwares are collected from web, so use at your own risk!

How To?

1. Download and Install Cactus Firmware Updater. ( Windows or Mac )

2. Download Domainlist.inc ( Original ), replace original one in your Cactus Firmware Updater directory.

( Default for Windows is C:\Program Files\Cactus Firmware Updater ).

Done! Open the updater and do the update!
